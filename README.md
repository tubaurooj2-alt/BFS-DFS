 Maze Editor & Solver (BFS / DFS)

An interactive **Maze Editor and Pathfinding Visualizer** built using pure HTML, CSS, and JavaScript.
This tool allows you to **create mazes, visualize algorithms, and understand how BFS and DFS work in real time**.
 Features

 Interactive grid-based maze editor
 Draw and erase walls with mouse drag
 Set custom **Start** and **End** points
 Visualize:

* BFS (Breadth-First Search) – finds shortest path
* DFS (Depth-First Search) – explores deeply

 Maze generator (auto-create random mazes)
 Real-time stats:

* Nodes explored
* Path length
* Time taken
* Wall density

 Sound effects for better experience
 Export & import maze as JSON

 How to Use

1. Select a tool:

   * Wall → Draw obstacles
   * Start → Place starting point
   * End → Place destination

2. (Optional) Click Generate to auto-create a maze

3. Choose algorithm:

   * BFS (shortest path)
   * DFS (not guaranteed shortest)

4. Click Solve

 Keyboard Shortcuts

| Key   | Action        |
| ----- | ------------- |
| Space | Solve         |
| G     | Generate maze |
| C     | Clear walls   |
| R     | Reset grid    |
| S     | Toggle sound  |
| 1     | Wall tool     |
| 2     | Start tool    |
| 3     | End tool      |

 Technologies Used

* HTML5 Canvas
* CSS (Modern UI styling)
* Vanilla JavaScript
* Web Audio API (for sound effects)


 Algorithms Explained

 🔵 BFS (Breadth-First Search)

* Explores level by level
* Always finds the **shortest path**

 🟣 DFS (Depth-First Search)

* Explores deeply first
* May not find the shortest path
